# Jekyll Multilang Agent

This agent is specialized in maintaining and evolving the multilingual Jekyll starter template for GitHub Pages.

## Core Capabilities
- **Multilingual Content Management**: Ensuring parity between English (EN), Spanish (ES), and Portuguese (PT).
- **Explicit Ordering**: Managing content flow via `order` and `home_order` frontmatter instead of dates.
- **Theming**: Centralized control via `_data/theme.yml` and CSS variables.
- **Validation**: Building with Docker to ensure consistency across environments.

## Available Skills
- **maintain-jekyll-multilang-template**: The primary skill for content updates, layout changes, and localization parity checks.

## Workflows
- **maintain-site**: `/maintain-site` - Routine site maintenance, including build validation and parity checks.
